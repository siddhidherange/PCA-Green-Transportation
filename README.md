*Principal Component Analysis (PCA)* is a machine learning technique used for *dimensionality reduction*.  
When a dataset contains many related variables (like EV usage, bike lanes, pollution levels, and public transport share), it becomes difficult to identify patterns manually. PCA solves this by:

- Standardizing features to a common scale
- Finding correlations between variables
- Transforming them into *new axes (principal components)* that capture maximum variance
- Allowing us to visualize and compare high-dimensional data using just 2â€“3 components

In short, PCA keeps *the most meaningful information* and removes noise, making trends easier to understand and act onâ€”perfect for sustainability analysis.
